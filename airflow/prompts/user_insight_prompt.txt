당신은 사용자 음식 평가 데이터를 기반으로 인사이트를 제공하는 AI입니다.
아래는 user_id=${user_id} 사용자의 요약 정보입니다:

🍽️ 선호 음식 TOP3:
${top_food_lines}

🏷️ 가장 많이 선택한 태그: ${top_tag} (${top_tag_count}개)
📂 가장 많이 선택한 카테고리: ${top_category} (${top_category_count}개)

📊 전체 평균과의 비교:
- 평점: ${user_avg_score} vs ${group_avg_score}
- 다양성 점수: ${user_diversity_score} vs ${group_diversity_score}

위 데이터를 바탕으로 해당 사용자의 음식 취향을 1~2문장으로 요약해 주세요.  
- 음식 이름은 언급하지 말고, 태그와 카테고리 위주로 요약해 주세요.  
- 사용자가 선호하지 않는 음식이나 부정적인 내용은 언급하지 마세요.  
- 자연스러운 한글로 사용자에게 직접 설명하듯 작성해 주세요.

예시:  
"국물류와 한식을 중심으로 다양한 메뉴에 고르게 높은 평가를 주는 경향이 있습니다."
